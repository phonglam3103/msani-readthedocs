About EirVS
******************

**EirVS** is a Python package for preparation (remove salts, tautomer standardization, stereoisomers enumeration, protonation) and filtering undesirable substructures (PAINS, reactive functional groups) for drug discovery projects. MolSanitizer is also capable of building DB2 files (for DOCK3.8) and PDBQT files (for AutoDock Vina) programs. Different conformational formats (SDF, Mol2) could also be generated by EirVS.

Check out the :doc:`usage` section for further information, including
how to :doc:`installation` the project.

.. note::

   This project is under active development.
   For any feedback, please refer to the :doc:`feedback` section.
